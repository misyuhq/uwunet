# Uwunet

Ce projet scan une plage d’adresses IP internes, détecte les machines où le port SSH (22) est ouvert, puis teste la connexion SSH uniquement sur des systèmes appartenant à l’utilisateur.
- `main.py`  
- `open_ssh.txt`

⚠️ Avertissement légal 
> Ce projet est fourni uniquement à des fins d’apprentissage.
> Scanner ou tenter d’accéder à des machines sans autorisation est strictement illégal.
> N’utilise ce script que dans ton propre réseau, sur des machines dont tu es propriétaire ou pour lesquelles tu as une autorisation explicite.
